# Library Management System

This is a Java Object-Oriented Programming project for managing books, members, loans, and returns in a library.

## Project Purpose
The system helps a library keep track of book lending and returning while enforcing proper business rules.

## Features
- Register members
- Add and manage books
- Issue books to members
- Return borrowed books
- Track loan records
- Handle invalid input gracefully

## Technologies Used
- Java
- NetBeans
